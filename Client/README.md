# Legacy Client Frontend

This folder contains an older static frontend for the shop project. It is kept in the repository for reference and includes:

- HTML pages for shop flows like home, products, cart, checkout, login, and register
- Shared CSS and JavaScript assets
- Local product seed data
- Page-specific scripts for the legacy implementation

## Status

This is not the active application entry point. The current app lives in the Next.js workspace under `src/`.

## If You Need To Work In This Folder

Use the files here only if you are maintaining or comparing the legacy frontend. For the active storefront, update the Next.js app instead.
