# Ecommerce Website

A full-featured online shopping website built with Node.js, Express, PostgreSQL, and functional Vanilla JS.

## Getting Started

1.  **Install Dependencies**

    ```bash
    npm install
    ```

2.  **Environment Setup**
    Copy `.env.example` to `.env` and fill in your database credentials.

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
